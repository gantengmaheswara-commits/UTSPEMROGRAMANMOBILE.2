## ✨ Fitur Utama
* **Splash Screen**: Branding awal aplikasi yang elegan dengan tema *Deep Navy*.
* **Login System**: Keamanan akses pengguna sebelum masuk ke dalam aplikasi.
* **AI Smart Analysis**: Konsultasi kesehatan tidur interaktif berbasis **Groq AI (Llama 3)**.
* **User Profile**: Manajemen informasi akun dan monitoring statistik kualitas tidur.

## 🛠️ Tech Stack & Tools
* **Core Language**: Kotlin
* **UI Framework**: XML Layout (ConstraintLayout)
* **AI Engine**: Groq Cloud SDK
* **Development**: Android Studio
* **Project Management**: ClickUP (Metode SCRUM)
